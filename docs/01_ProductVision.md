# Product Vision
## めざせ！名監督｜野球IQアカデミー

## 1. Product Concept
少年野球の子どもが、試合中の状況判断をゲームで学ぶスマートフォン向けWebゲーム。AI監督が理解度を見ながら次の練習を提案し、成長を伴走する。

## 2. Primary Users
### 子ども
小学校低学年を中心とする少年野球選手。野球経験が浅く、試合中の判断に迷う子。

### 保護者
野球ルールに詳しくなくても、子どもの成長を確認したい人。

### 指導者
選手やチームの理解度を把握し、練習メニュー選定に役立てたい人。

## 3. Core Experience
1. AI監督が今日のミッションを出す
2. 子どもが状況を見て判断する
3. 正誤だけでなく理由を知る
4. 理解度が更新される
5. AI監督が次の練習を提案する
6. ランク、図鑑、バッジが成長を示す

## 4. Game Modes
- 守備編：どの塁でアウトを取るか、誰がどこへ動くか
- ランナー編：進む、戻る、止まるの判断
- バッター編：将来追加

## 5. Learning Themes
- base_order：塁の順番、進塁方向
- basic_advance：基本的な進塁
- force_out：フォースアウト
- tag_out：タッチアウト
- return_to_base：帰塁
- multiple_runners：複数ランナーの判断
- tag_up：タッチアップ
- defensive_choice：守備側がアウトを取る塁の選択

## 6. Rank Structure
Lv1 体験生 / Lv2 入団 / Lv3 準レギュラー / Lv4 レギュラー / Lv5 オール上尾 / Lv6 コーチ / Lv7 監督 / Lv8 NPB / Lv9 メジャー / Lv10 殿堂入り

Lv4「レギュラー」を小学校低学年の基本卒業ラインとする。

## 7. Main Screens
- タイトル
- ホーム
- ゲーム
- AI監督
- 図鑑
- 設定
- 保護者・指導者画面（将来）

## 8. Character and World
FUJICON独自の少年野球世界を作る。明るい球場、親しみやすい監督、紺・白・赤を基調としたユニフォーム。子ども向けだが幼稚すぎない。既存作品を模倣しない。

## 9. Success Criteria
- 子どもが説明なしで1問目を始められる
- 1セッションを完走できる
- 終了後にもう一度遊びたいと思える
- 間違えた理由を理解できる
- 次に何を練習すべきか分かる
- 実際の練習や試合で学びを思い出せる

## 10. Early Non-goals
オンラインチャット、不特定多数ランキング、自由投稿、クラウドアカウント、対戦型生成AI、大規模3D、Nintendo Switch本体配信は初期版では優先しない。
