# 野球IQアカデミー 2.0β

「開いた瞬間に面白そうと思えること」を優先したデザインβ版です。

## 実装内容
- 承認済みの少年野球イラストを実ゲーム素材として使用
- ホーム画面
- ステージ選択
- ルール学習
- 守備の基本5問
- ヒント
- 正誤解説
- XP
- 結果画面

## 更新方法
ZIPを展開し、中身をすべてGitHub Desktopのローカルフォルダ
`-baseball-iq-game`
へ上書きします。

`assets`フォルダも必ずコピーしてください。
