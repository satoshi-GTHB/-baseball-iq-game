# 野球IQアカデミー Ver4.0α.1

## 修正方針
レベルは累積XPだけでは決まりません。各レベルの問題難易度と理解度を組み合わせて昇格します。

- 問題ごとに difficulty（1〜4）を設定
- 現在のレベルに対応する問題を出題
- 昇格条件は「必要XP」「必要回答数」「理解度」の3条件
- Lv.4 レギュラーを小学校低学年の到達目標に設定
- Lv.5 オール上尾以降は発展カリキュラムとしてβ版以降に追加

## 現在のカリキュラム
1. 体験生：塁の順番、打者走者の基本
2. 入団：1人のランナーとフォースアウトの基本
3. 準レギュラー：進塁・帰塁、前後の打球判断
4. レギュラー：複数ランナー、満塁を含む状況判断
